# Movve
## Description
Приложение, показывающее популярные сегодня фильмы и сериалы с возможностью посмотреть подробности,
добавление в избранное и поиск.
## Architecture
VIPER
## Frameworks
- UIKit
- Alamofire
- Kingfisher
- RealmSwift
- youtube-ios-player-helper-swift

## Setup
```
$ cd .../Movve/
$ pod install
```
